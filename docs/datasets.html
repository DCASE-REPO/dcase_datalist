Table notation

Audio content type:

  • Freefield, audio captured in real uncontrolled environment
  • Synthetic, audio created synthetic means by mixing audio from various sources
  • Isolated, audio recorded in controlled environment where only single sound source is active at time

Scene content:

  • Constant
  • Variable

Recording setup:

  • Near-field
  • Far-field
  • Mixed
  • Uncontrolled
  • Unknown

Recording spot type:

  • Fixed, microphone spot is fixed during the recording
  • Moving, microphone is moving during the recording
  • Unknown

Material source:

  • Original, audio captured originally for the dataset by the data collectors
  • Online, audio collected from online sources (other than Youtube or Freesound)
  • Youtube, audio collected from youtube.com
  • Freesound, audio collected from freesound.org
  • Crowdsourced, audio collected crowsourced manner, data collected e.g. by contributing users

Annotation type:

  • Weak, sound event class annotation done per audio recording
  • Strong, sound event class annotation done with exact onsets and offsets
  • None, no annotation provided

Annotation source:

  • Experts
  • Crowdsourced
  • Synthetic
  • Location

License type:

  • Free, free for academic usage (non-commercial), usually released under university specific EULA
  • Creative Commons, released under creative commons licence
  • Commercial, commercial license
  • Proprietary, dataset has not been published, but there is publications reporting results using it
  • R&D only, dataset is allowed only for research and development use
